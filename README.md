# Employee Task Management System

## Project Overview
A full-stack Employee Task Management System built for assignment evaluation.

---

## Tech Stack

### Frontend:
- React + TypeScript
- Redux Toolkit
- React Router
- Axios

### Backend:
- Node.js
- Express.js
- JWT Authentication
- Multer (File Upload)

### Database:
- MySQL
- 
## Features

### Authentication
- JWT Login/Register
- Remember Me
- Role-based access (Admin / Employee)

### Admin Features
- Employee CRUD (Create, Read, Update, Delete)
- Task CRUD
- Assign tasks to employees
- Reports (Completed, Pending, Employee-wise)

### Employee Features
- View only assigned tasks
- Update task status
- My Tasks dashboard

### Other Features
- File Upload (PDF, JPG, PNG)
- Search, Sort, Pagination
- Protected Routes
